```
┌─────────────────────────────────────────────────────────────┐
│  Пользователь входит в аккаунт                              │
│  ↓                                                           │
│  Loader проверяет подписку через /api/auth/check            │
│  ↓                                                           │
│  Игра запускается                                           │
│  ↓                                                           │
│  Каждую секунду: Loader → /api/auth/verify-subscription    │

```
