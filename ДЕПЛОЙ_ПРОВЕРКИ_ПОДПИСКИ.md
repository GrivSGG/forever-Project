# Деплой системы проверки подписки в реальном времени

## ✅ Что уже сделано

### 1. API обновлен
- Добавлен endpoint `/api/auth/verify-subscription`
- Проверяет подписку быстро (для периодических запросов)
- Автоматически деактивирует истекшие подписки

### 2. Loader обновлен и собран
- Добавлена функция `VerifySubscription()`
- Добавлен поток `SubscriptionCheckThread()`
- Проверка каждую секунду во время игры
- Автоматическое завершение при отзыве подписки
- **Файл:** `D:\cataclysm\loader\ForeverLoader.exe` (539 KB, обновлен 29.03.2026 0:19:34)

## 📋 Что нужно сделать

### Шаг 1: Задеплоить API на Railway

```bash
cd forever-auth-api
git add server.js
git commit -m "Add real-time subscription verification endpoint"
git push
```

Railway автоматически задеплоит изменения.

### Шаг 2: Проверить деплой

Подождите 1-2 минуты и проверьте:
```bash
curl -X POST https://forever-project-production-a2fa.up.railway.app/api/auth/verify-subscription \
  -H "Content-Type: application/json" \
  -d '{"username":"admin"}'
```

Ожидаемый ответ:
```json
{
  "success": true,
  "active": true,
  "type": "lifetime",
  "expiresAt": null,
  "betaAccess": true
}
```

### Шаг 3: Загрузить новый loader на GitHub

1. Переименуйте старый loader (если нужен бэкап):
   ```bash
   # В репозитории forever-Project
   mv ForeverLoader.exe ForeverLoader-old.exe
   ```

2. Скопируйте новый loader:
   ```bash
   copy D:\cataclysm\loader\ForeverLoader.exe <путь_к_репозиторию>\ForeverLoader.exe
   ```

3. Закоммитьте и запушьте:
   ```bash
   git add ForeverLoader.exe
   git commit -m "Add real-time subscription verification to loader"
   git push
   ```

## 🧪 Тестирование

### Тест 1: Отзыв подписки во время игры

1. Создайте тестового пользователя с подпиской "7days"
2. Запустите игру через loader
3. Откройте helper панель
4. Отзовите подписку у пользователя
5. **Ожидаемый результат:** Игра завершится в течение 1 секунды с сообщением

### Тест 2: Истечение подписки

1. Создайте подписку "1day"
2. В Firebase измените `expiresAt` на прошлую дату
3. Подождите 1 секунду
4. **Ожидаемый результат:** Игра завершится автоматически

### Тест 3: Бан пользователя

1. Запустите игру
2. Забаньте пользователя через helper панель
3. **Ожидаемый результат:** Игра завершится в течение 1 секунды

## ⚠️ Важные замечания

1. **Loader обязательно нужно обновить** - старая версия не будет проверять подписку
2. **API должен быть задеплоен** - иначе loader не сможет проверить подписку
3. **Пользователи должны скачать новый loader** - автообновления нет
