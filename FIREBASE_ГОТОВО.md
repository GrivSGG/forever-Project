## 🚀 Как это работает:

### Регистрация пользователя:
1. Пользователь регистрируется на сайте
2. Данные сохраняются в localStorage (для офлайн режима)
3. Данные автоматически отправляются в Firebase Firestore
4. Хелпер панель получает обновление в реальном времени

### Создание тикета:
1. Пользователь создает тикет
2. Тикет сохраняется в localStorage
3. Тикет автоматически отправляется в Firebase
4. Хелпер видит новый тикет мгновенно

### Панель хелпера:
- Все данные загружаются из Firebase
- Обновления приходят в реальном времени (без перезагрузки)
- Можно отвечать на тикеты
- Можно менять статусы тикетов
- Можно генерировать ключи
- Можно банить пользователей
