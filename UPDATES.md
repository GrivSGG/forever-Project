## 📋 Как использовать:

### Вход как хелпер:
1. Откройте главную страницу
2. Нажмите "Войти"
3. Введите:
   - Логин: `Helper`
   - Пароль: `MZFeEgJY`
4. Вы будете перенаправлены в панель хелпера

### Создание тикета (пользователь):
1. Войдите в свой аккаунт
2. Откройте личный кабинет
3. Найдите кнопку "Создать тикет" или "Поддержка"
4. Заполните форму:
   - Проблема: краткое описание
   - Описание: подробное описание
5. Нажмите "Отправить тикет"

### Ответ на тикет (хелпер):
1. Войдите как хелпер
2. Откройте раздел "Тикеты"
3. Выберите тикет
4. Напишите ответ
5. Нажмите "Отправить"

### Блокировка пользователя (хелпер):
1. Войдите как хелпер
2. Откройте раздел "Управление банами"
3. Введите логин пользователя
4. Укажите длительность (минуты, 0 = навсегда)
5. Укажите причину
6. Нажмите "Заблокировать"

### Генерация ключей (хелпер):
1. Войдите как хелпер
2. Откройте раздел "Генератор ключей"
3. Выберите тип лицензии
4. Укажите количество
5. Добавьте комментарий (опционально)
6. Нажмите "Сгенерировать"
