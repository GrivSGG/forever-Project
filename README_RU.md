## ⚠️ Что нужно добавить

Добавьте 3 файла в папку `assets/`:

```
assets/
├── logo.png          ← Ваш логотип (200x200px)
├── background.mp4    ← Видео для фона (Full HD, до 10MB)
└── favicon.ico       ← Иконка сайта (32x32px)
```

## 🐛 Решение проблем

### Видео не проигрывается:
- Проверьте, что файл `background.mp4` существует
- Проверьте формат (MP4)
- Проверьте размер (не более 10MB)

### Логотип не отображается:
- Проверьте путь: `assets/logo.png`
- Проверьте формат (PNG или JPG)

### Статистика показывает 0:
- Зарегистрируйте тестовые аккаунты
- Обновите страницу

### Email не отправляются:
- Это нормально! Сейчас работает симуляция
- Проверьте console.log в браузере (F12)
- Для реальной отправки нужен backend
