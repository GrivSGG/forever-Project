🔧 Исправление прав доступа

⚠️ Проблема: Helper панель показывает "Missing or insufficient permissions"

Причина: Firestore Rules блокируют чтение данных

Шаг 1: Откройте Firebase Console

Перейдите по ссылке:

Открыть Firebase Console

Шаг 2: Скопируйте правила

Скопируйте эти правила (нажмите на блок чтобы выделить):

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read: if true; allow write: if false; } match /tickets/{ticketId} { allow read: if true; allow write: if false; } match /licenses/{licenseId} { allow read: if true; allow write: if false; } match /bans/{banId} { allow read: if true; allow write: if false; } match /grantHistory/{historyId} { allow read: if true; allow write: if false; } match /{document=**} { allow read, write: if false; } } }

Шаг 3: Вставьте и опубликуйте

1. Удалите старые правила в Firebase Console

2. Вставьте скопированные правила

3. Нажмите кнопку "Опубликовать" (Publish)

Шаг 4: Подождите 1 минуту

Правила применяются не мгновенно. Подождите 1-2 минуты.

Шаг 5: Обновите helper панель

Нажмите F5 или перейдите заново:

Войти в Helper Panel
✅ После обновления правил:
  • Чтение данных будет разрешено
  • Запись только через API (безопасно)
  • Helper панель заработает полностью
