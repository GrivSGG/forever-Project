## 🎨 Настройка

### Изменить YouTube видео:
Откройте `index.html` и найдите:
```html
<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ"
```
Замените `dQw4w9WgXcQ` на ID вашего видео

### Изменить цены:
Откройте `index.html` и найдите секцию `#pricing`

### Изменить цвета:
Откройте `style.css` и измените переменные в `:root`

## 🐛 Решение проблем

### Видео не проигрывается:
- Проверьте, что файл `background.mp4` существует
- Проверьте формат видео (должен быть MP4)
- Проверьте размер файла (не более 10MB)

### Логотип не отображается:
- Проверьте путь: `assets/logo.png`
- Проверьте формат (PNG или JPG)
- Проверьте размер (рекомендуется 200x200px)

### Статистика показывает 0:
- Зарегистрируйте несколько тестовых аккаунтов
- Статистика обновляется автоматически
- Данные хранятся в localStorage браузера

### Email не отправляются:
- Это нормально! Сейчас работает симуляция
- Для реальной отправки нужен backend сервер
- Проверьте console.log в браузере

## 📊 Статистика

### Где хранятся данные:
- `localStorage.forever_users` - пользователи
- `localStorage.forever_session` - текущая сессия
- `localStorage.forever_downloads` - счетчик скачиваний
- `localStorage.deviceId` - ID устройства

### Как очистить данные:
```javascript
// В консоли браузера (F12)
localStorage.clear()
```
