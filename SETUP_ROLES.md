# 👥 Настройка ролей пользователей

## Как дать права админа или хелпера

### Способ 1: Через Firebase Console (Рекомендуется)

1. Откройте Firebase Console: https://console.firebase.google.com/
2. Выберите проект "forever-client"
3. Перейдите в Firestore Database
4. Найдите коллекцию `users`
5. Найдите нужного пользователя
6. Нажмите на документ
7. Добавьте или измените поле `role`:
   - Для админа: `role: "admin"`
   - Для хелпера: `role: "helper"`
   - Для обычного пользователя: `role: "user"` (или удалите поле)

### Способ 2: Через Helper Panel

1. Войдите в Helper Panel как админ
2. Перейдите в раздел "Пользователи"
3. Найдите нужного пользователя
4. Измените его роль

### Текущие роли

**admin** - Полный доступ:
- ✅ Админ панель
- ✅ Helper панель
- ✅ Создание/удаление пользователей
- ✅ Генерация ключей
- ✅ Управление банами
- ✅ Просмотр всех данных

**helper** - Ограниченный доступ:
- ✅ Helper панель
- ✅ Просмотр пользователей
- ✅ Ответы на тикеты
- ✅ Генерация ключей
- ✅ Управление банами
- ❌ Админ панель

**user** (или без роли) - Базовый доступ:
- ✅ Личный кабинет
- ✅ Активация лицензий
- ✅ Создание тикетов
- ❌ Админ панель
- ❌ Helper панель

## Первоначальная настройка

### Шаг 1: Создайте админа

Зарегистрируйте аккаунт на сайте, затем через Firebase Console добавьте ему роль:

```
Коллекция: users
Документ: ваш_логин
Поле: role
Значение: "admin"
```

### Шаг 2: Создайте хелпера

Аналогично, но с ролью "helper":

```
Коллекция: users
Документ: логин_хелпера
Поле: role
Значение: "helper"
```

### Шаг 3: Настройте Security Rules

Скопируйте правила из `FIRESTORE_RULES_SECURE.txt` в Firebase Console.
