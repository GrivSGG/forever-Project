# Система лицензий Forever Client

## Для администратора

### Генерация ключей

1. Открой файл `keygen.html` в браузере
2. Выбери тип лицензии
3. Введи имя пользователя (опционально)
4. Нажми "Сгенерировать ключ"
5. Скопируй ключ и отправь покупателю

### Форматы ключей

- `FC30-XXXX-XXXX-XXXX-XXXX` - 30 дней (199₽)
- `FC90-XXXX-XXXX-XXXX-XXXX` - 90 дней (450₽)
- `FCLT-XXXX-XXXX-XXXX-XXXX` - Навсегда (600₽)
- `FCHW-XXXX-XXXX-XXXX-XXXX` - Сброс HWID (199₽)
- `FCBT-XXXX-XXXX-XXXX-XXXX` - Бета доступ (1000₽)

### История ключей

Все сгенерированные ключи сохраняются в localStorage браузера.
Чтобы посмотреть историю, открой консоль браузера (F12) и введи:
```javascript
JSON.parse(localStorage.getItem('keyHistory'))
```

## Для пользователей

### Активация лицензии

1. Запусти Minecraft с Forever Client
2. В главном меню нажми кнопку "Активировать лицензию"
3. Введи полученный ключ
4. Нажми "Активировать"

### Проверка лицензии

Лицензия проверяется при каждом запуске чита.
Информация о лицензии отображается в главном меню.

### Сброс HWID

Если нужно перенести лицензию на другой компьютер:
1. Купи ключ "Сброс HWID"
2. В меню активации введи этот ключ
3. Лицензия будет привязана к новому компьютеру

## Безопасность

### HWID (Hardware ID)

Каждая лицензия привязывается к уникальному идентификатору компьютера (HWID).
HWID генерируется на основе:
- Операционной системы
- Архитектуры процессора
- Имени пользователя
- Серийного номера процессора

### Хранение лицензии

Лицензия хранится в файле `forever_license.dat` в папке с игрой.
Файл зашифрован и содержит:
- Ключ лицензии
- Тип лицензии
- HWID
- Дату активации
- Дату истечения

## Интеграция с сайтом

### Обновление сайта

После изменения цен или добавления новых тарифов:

```bash
cd forever-website
git add .
git commit -m "Обновление цен"
git push
```

Изменения появятся на сайте через 1-2 минуты.

### Добавление способов оплаты

Отредактируй файл `index.html`, секцию "Контакты":
- Добавь ссылки на Discord/Telegram
- Укажи реальные способы оплаты
- Добавь инструкции по покупке

## Расширенные возможности

### Онлайн проверка лицензий

Для более надежной защиты можно добавить онлайн проверку:

1. Создай базу данных с ключами
2. Настрой API для проверки ключей
3. В `LicenseManager.java` раскомментируй код проверки через API

### Автоматическая выдача ключей

Можно интегрировать с платежными системами:
- QIWI API
- ЮMoney API
- Telegram Bot для автоматической выдачи

### Статистика продаж

Добавь в `keygen.html` отправку данных о сгенерированных ключах на сервер для учета продаж.
